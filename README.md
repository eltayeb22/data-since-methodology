#  Crypto Data Analysis & Methodology

A Python project for extracting, cleaning, and analyzing Cryptocurrency data using APIs.

##  Project Contents
* `gui.py`: The main graphical user interface for the app.
* `Crypto(1).ipynb`: Data exploration and analysis notebook.
* `Model.joblib`: Pre-trained model for data predictions.
* `requirements.txt`: List of Python libraries needed (pandas, requests, etc.).

## How to use
1. Install dependencies: `pip install -r requirements.txt`
2. Run the application: `python gui.py`
